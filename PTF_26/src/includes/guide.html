Components Guide

Button

버튼 사용 가이드: 기본 버튼은 .btn 클래스를 사용.

  • 크기: .btn-sm, .btn-md, .btn-lg
  • 스타일: .btn-primary (주요), .btn-secondary (보조), .btn-outline (테두리)
  • 접근성: 버튼에는 명확한 텍스트 라벨을 사용하고 필요 시 aria-label을 추가.
    폼 제출 버튼은 type=\"submit\"을, 일반 버튼은 type=\"button\"을 명시 할 것.

예시:
<button class="btn btn-primary btn-sm" type="button">확인</button>

Label

라벨 사용 가이드: 기본 라벨은 .label-txt 클래스를 사용.

  • 기본 라벨
  • 주요 라벨
  • 구분 라벨
  • 구분 라벨
  • 구분 라벨
  • 스타일: .is-default (기본), .is-primary (주요), .is-success (성공), .is-warning (경고), .is-danger (위험)
  • 접근성:라벨에는 명확한 텍스트 라벨을 사용하고 필요 시 aria-label을 추가.

예시:
<span class="label-txt is-primary" aria-label="주요 라벨">주요 라벨</span>

Icon

아이콘 사용 가이드: 기본 아이콘은 .ico , ic-아이콘명, ic-s-사이즈변수 클래스를 사용.

  • 아이콘 모양: .ic-info (정보), .ic-x (닫기), .ic-search (검색), .ic-setting (설정)
  • 아이콘 크기: .ic-s-sm (작게), .ic-s-md (중간), .ic-s-lg (크게)
  • 접근성: 아이콘에는 명확한 텍스트 라벨을 사용하고 필요 시 aria-label을 추가.

예시:
<span class="ico ic-info" aria-label="정보 아이콘"></span>
